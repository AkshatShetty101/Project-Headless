About Us

This website is an interface to search the Indian Ecourt website for all district court based cases. It offers an easy and client friendly feel to the whole procedure. It takes out the effort and time required to access the site and make individual searches for each year and each court by pipelining the whole operation. It offers large amount of error handling while providing seamless transition from page to page.
